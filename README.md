# Trabalho 1: Realizado por Guilherme Eduardo & Heloisa Pinho

## Requisitos

- Misturar uma **técnica de substituição** com uma **técnica de transposição** para criar um algoritmo criptográfico.  
  - Pode usar técnicas já existentes, criar variações ou ideias novas.  
  - Pode executar cada técnica uma vez ou mais vezes.
  - Cada estudante deve usar cada técnica **apenas uma vez**.  
  - Implementar o código para **cifrar e decifrar arquivos de texto** em **UTF-8**.  

## Comparações

- Comparar a velocidade de **cifrar** e de **decifrar** do algoritmo criado com a implementação do **AES** da biblioteca **OpenSSL**.  
  - As comparações devem ser feitas com **3 livros do projeto Gutenberg.**:  
    - Um arquivo com menos de **10 KB**  
    - Um arquivo com mais de **100 K**  
    - Um arquivo com mais de **1 MB**

- Comparar:  
  - Velocidade para **cifrar**  
  - Velocidade para **decifrar**  

## Implementação

- Pode ser feita em **qualquer linguagem de programação**.  
- Trabalho em **dupla**.  

## Entrega

- Um **relatório** com detalhes da cifra e da implementação.  
- Incluir **gráficos de barras** comparando os resultados com o **AES**.
